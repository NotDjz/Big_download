# BIG DL

Telechargeur standalone — YouTube, Instagram, TikTok, X/Twitter, SoundCloud.

## Utilisation

Lancer `BigDownloader.exe`. Coller un lien, choisir le format, telecharger.
Onglet "Decouper" pour couper un fichier audio/video local.

## Cookies Instagram (photos uniquement)

Les photos Instagram necessitent un fichier cookies. Exporter les cookies avec
l'extension "Get cookies.txt LOCALLY" et placer `cookies.txt` a cote de l'exe.
Les reels/videos Instagram fonctionnent sans cookies.
